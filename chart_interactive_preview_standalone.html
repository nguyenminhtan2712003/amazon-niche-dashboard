📐 Architecture: Cách hoạt động trong Dashboard thật

Mỗi lần click niche, Next.js gọi 1 API endpoint duy nhất /api/db/niches/[id]/detail → trả về JSON gộp cả 6 chart (như preview này) → React state cập nhật → tất cả chart re-render. Latency: ~150-300ms nhờ DuckDB columnar query.

🟢 Aggregate Mode (no selection)

  • Chart #1, #5, #6, #9, #10 (overview)
  • Đứng yên, pre-computed
  • Load 1 lần khi vào page

🟣 Hybrid Mode

  • Chart #2, #4, #14, #15, #16, #18
  • Mặc định show aggregate
  • Khi có niche selected → filter sang category của niche đó

🔵 Detail Mode (niche selected)

  • 6 chart A-F trong preview này
  • Chỉ hiển thị khi đã chọn niche
  • Tính ON-DEMAND qua DuckDB
